看懂運作原理:


這支程式的核心就是：
1. 用滑鼠座標（拖曳時的 clientX/clientY）計算目前方向和距離
2. 根據方向決定哪個按鈕要高亮（顯示變化）
3. 在放開滑鼠時，依目前高亮的按鈕執行對應動作（搜尋、複製、翻譯）

簡單講就是「座標 -> 視覺狀態 -> 行為觸發」這條流程。

另外它還有兩個保護區域：
1. 太靠近中心（DEAD_ZONE）不觸發
2. 太遠（超過 SENSITIVITY）也不觸發

所以不是只看方向，也會看距離，避免誤觸。**
